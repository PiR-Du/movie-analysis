# Movie analysis

## Objectif
Analyser et visualiser des données de films pour découvrir des tendances : genres populaires, notes IMDb, évolution des films dans le temps.

## Contenu
- **data/** : fichiers CSV/JSON téléchargés depuis IMDb/TMDB
- **notebooks/** : exploration et visualisation des données
- **scripts/** : scripts Python pour récupérer et transformer les données
- **requirements.txt** : dépendances Python

## Méthodes
- Nettoyage et transformation des données
- Analyse descriptive
- Visualisation avec matplotlib & seaborn
